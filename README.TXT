Projekt: Online-Shop-JUnit
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 5. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2012

Dieses Projekt implementiert einen kleinen Teil eines Online-Shop-Systems (wie
z.B. Amazon.com). Das aktuelle Projekt konzentriert sich ganz auf die Kommentare, 
die die Kunden für die Produkte abgeben. Es enthält Code zum Erzeugen, Speichern, Anzeigen
und Bearbeiten von Kundenkommentaren.

Ziel: Einführung in die Durchführung von Regressionstest mit JUnit. Das Projekt enthält
eine Testklasse, die einen Teil der Testfunktionalität demonstriert. Weitere Erläuterungen 
zum Testen finden Sie im Buch.

HINWEIS: In diesen Klassen sind verschiedene, ernste Fehler enthalten.
